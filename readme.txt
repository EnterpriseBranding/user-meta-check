Shortcode to hide content based on user meta value.

== Description ==

Add shortcodes like this to your post to hide content from users based on their user meta values:

[user-meta-check key='last_name' value='Coleman']This content is only shown to users with the last name 'Coleman'[/user-meta-check]

== Installation ==

= Prerequisites =
1. You must have Paid Memberships Pro installed and activated on your site.

= Download, Install and Activate! =
1. Download the latest version of the plugin.
1. Unzip the downloaded file to your computer.
1. Upload the /user-meta-check/ directory to the /wp-content/plugins/ directory of your site.
1. Activate the plugin through the 'Plugins' menu in WordPress.
